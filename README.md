# Лабораторная работа по мвтематическому анализу-2
Проект, посвящённый практическому применению методов математического анализа, численных вычислений и обработки сигналов на Python. В работе объединены задачи по рядам Фурье, дискретному преобразованию Фурье, кратным интегралам и анализу изображений.

В ноутбуке рассматриваются аппроксимация функций с помощью рядов Фурье, фильтрация аудиосигналов через DFT, численное моделирование освещённости с использованием кратных интегралов, а также задачи по Scale-Invariant Feature Transform (SIFT). Для вычислений и визуализации используются NumPy, SymPy, SciPy, Matplotlib, OpenCV, scikit-learn и дополнительные инструменты для работы со звуком и изображениями.

Проект показывает, как методы математического анализа применяются в задачах обработки сигналов, численного моделирования, компьютерного зрения и вычислительной математики.
